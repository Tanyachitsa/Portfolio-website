Python recipe search

A program where the user searches for recipes based an ingredient and this function makes a request to the API and it returns recipes from the API response, which are displayed. For the extention, the results displayed were saved to a text file.

Python


import requests
requests

pprint

url = 'https://api.edamam.com/search?q={}&app_id={}&app_key={}'.format(ingredient, app_id, app_key)
response = requests.get(url)
print(response)
recipe = response.json()
pprint(recipe)
